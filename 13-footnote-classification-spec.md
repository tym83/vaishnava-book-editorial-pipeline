# Footnote Classification Spec v1

## 1. Назначение

Этот слой нужен для книг, где одного общего стиля `footnote text` недостаточно и сноски нужно разнести по 4 каноническим стилям.

---

## 2. Рабочая семантика типов

### `Сноска 1`

Пояснительная / словарная / контекстная сноска.

Примеры:

- краткое объяснение термина;
- историческая или географическая справка;
- пояснение имени, титула, предмета.

### `Сноска 2`

Ссылочная / библиографическая / цитатная сноска.

Примеры:

- `Чч 1.1.20-21`
- `Гв 2.169-70`
- `Письмо (от 25 июня 1929 года), «Патравали» 2.68`
- краткая ссылка на книгу, том, главу, страницу, источник.

### `Сноска 3`

Сноска переводчика.

Признак:

- явное `прим. переводчика`

### `Сноска 4`

Перекрестная / редакторская / навигационная сноска.

Примеры:

- `см. в томе 1`
- `смотри комментарий`
- `подробное описание можно найти ...`
- `данный случай описан в ...`
- `прим. ред.`
- редакционные указания на доступность русского издания, сайта, параллельного материала и т.п.

## 4. Текущее состояние эвристик

Хорошо ловятся:

- явные ссылки на писания и источники;
- письма/лекции с датами и ссылками;
- `прим. переводчика`;
- `см.`, `смотри`, `в томе`, `описан в`, `по аналогичной схеме`, `прим. ред.`

Консервативно остаются в `Сноска 1`:

- длинные пояснительные заметки;
- смешанные пояснительно-ссылочные заметки;
- редкие нестандартные формы.

Для неоднозначных мест используется `review` и `hints`.

## 6. Что еще не закрыто

1. Семантика `Сноска 4` пока широкая:
   - сюда попадают и перекрестные ссылки,
   - и редакторские/навигационные заметки.

2. Для части книг может понадобиться:
   - разделить `прим. ред.` и cross-reference;
   - выделить дополнительный подтип внутри `Сноска 1`.

3. Окончательная стабильность будет после ручной разметки нескольких десятков реальных сносок из:
   - `Вайбхавы`
   - `Шри Чайтаньи`
   - `Воспоминаний`
