# QA Checklists v1

## 1. Checklist: подготовка книги

- [ ] Источник определен: EN, RU или оба
- [ ] Формат источника определен
- [ ] Книга разделена на главы по H1
- [ ] Имена файлов приведены к `001`, `002`, ...
- [ ] Предисловие / введение / приложения выделены как отдельные единицы
- [ ] Все входные файлы доступны
- [ ] Нет OCR-катастрофы

---

## 2. Checklist: перевод главы

- [ ] H1 корректно выделен
- [ ] Подзаголовки сохранены
- [ ] Все абзацы переведены
- [ ] Шлоки проверены по `vedabase` или отмечены для сверки
- [ ] Перевод шлок выделен отдельно
- [ ] Письма отличены от обычных цитат
- [ ] Подписи к иллюстрациям переведены
- [ ] Названия СМИ не переведены без основания
- [ ] Названия журналов, газет, брендов и аналогичных объектов сохранены в оригинале, если нет утвержденного исключения
- [ ] Термины сверены с корпусом/глоссарием
- [ ] Сомнительные места помечены `UNCERTAIN`
- [ ] Есть секция `Translator Notes`

---

## 3. Checklist: полнота перевода

- [ ] Нет пропущенных абзацев
- [ ] Нет потерянных заголовков
- [ ] Нет потерянных списков
- [ ] Нет потерянных подписей
- [ ] Нет потерянных источников
- [ ] Нет потерянных писем
- [ ] Нет потерянных сносок
- [ ] Missing fragments оформлены как comments + suggested insertion
- [ ] Extra fragments не удалены автоматически, а маркированы

---

## 4. Checklist: смысловая точность

- [ ] Нет смысловых упрощений
- [ ] Нет потери полемического акцента
- [ ] Нет сглаживания богословских терминов
- [ ] Нет светских замен там, где нужен специальный термин
- [ ] Логические выводы сохранены
- [ ] Противопоставления сохранены
- [ ] Причинно-следственные связи сохранены
- [ ] Все замечания снабжены исправленным вариантом

---

## 5. Checklist: русская стилистика

- [ ] Русский текст читается естественно
- [ ] Нет явной кальки с английского
- [ ] Порядок слов естественный
- [ ] Длинные предложения разбиты там, где нужно
- [ ] Нет чрезмерного канцелярита
- [ ] Нет ненужного высокого штиля
- [ ] Сохранен тон автора
- [ ] Стиль главы ровный
- [ ] `Вы` не капитализировано по устаревшей привычке без особого основания

---

## 6. Checklist: корректура

- [ ] Орфография проверена
- [ ] Пунктуация проверена
- [ ] `ё` везде проставлена
- [ ] Кавычки приведены к русской норме
- [ ] Дефисы и тире выровнены
- [ ] Неразрывные пробелы проставлены
- [ ] Интервалы между числами и обозначениями проверены
- [ ] Латиница вместо кириллицы найдена
- [ ] Legacy-диакритика заменена на Unicode
- [ ] Термины и имена сверены
- [ ] Склонение проверено
- [ ] Диакритика снята в прозе там, где она не нужна
- [ ] Диакритика сохранена в шлоках, цитатах и разрешенных inline-цитатах
- [ ] Стандартные термины вроде `вани`, `вапу` не оставлены с лишней диакритикой

---

## 7. Checklist: структура и стили

- [ ] Все главы имеют `Заголовок 1`
- [ ] Иерархия заголовков соблюдена
- [ ] Шлоки размечены стилем `Шлока`
- [ ] Перевод шлок размечен стилем `Перевод шлоки`
- [ ] Цитаты 1 и 2 не смешаны
- [ ] `Шлока в цитате` используется там, где нужно
- [ ] Письма размечены стилем `Письмо`
- [ ] Источники размечены стилем `Источник`
- [ ] Подписи к иллюстрациям имеют отдельный стиль
- [ ] Inline-цитаты, длинные цитатные блоки и полноценные произведения не смешаны в одну схему оформления
- [ ] Использованы нужные типы сносок
- [ ] Основной текст везде оформлен единообразно
- [ ] Курсив и полужирный по возможности заданы символьными стилями
- [ ] Style overrides найдены и отмечены

---

## 8. Checklist: перед импортом в InDesign

- [ ] Названия Word-стилей совпадают с InDesign-стилями
- [ ] Нет критичных style overrides
- [ ] Локальный курсив сохранен
- [ ] Локальный полужирный сохранен
- [ ] Сноски сохранены
- [ ] Нет мусорного ручного форматирования
- [ ] Нет legacy-символов
- [ ] Структура документа не сломана

---

## 9. Checklist: после импорта в InDesign

- [ ] Текст импортирован полностью
- [ ] Нет overset text
- [ ] Все стили примапились корректно
- [ ] Заголовки не потерялись
- [ ] Списки не сломались
- [ ] Сноски перенеслись
- [ ] Курсив и полужирный сохранились
- [ ] Добавлены страницы, если текста не хватило
- [ ] Картинки не мешают тексту

---

## 10. Checklist: InDesign layout QA

- [ ] Переносы расставлены автоматически
- [ ] Неразрывные пробелы проверены
- [ ] Нет строк, которые начинаются с повисшего тире внутри абзаца
- [ ] Widows/orphans отмечены
- [ ] Missing fonts отмечены
- [ ] Style overrides отмечены
- [ ] Wrong language assignment отмечен
- [ ] Hyphenation settings согласованы
- [ ] Переносы включены и в сносках
- [ ] Footnote problems отмечены
- [ ] Legacy-символы найдены и отмечены
