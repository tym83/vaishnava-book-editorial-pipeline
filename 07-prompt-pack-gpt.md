# Prompt Pack v1 для GPT

## 1. Общая настройка

Использовать для GPT те же редакционные принципы, что и для Claude, но с более жесткой структурой вывода.

Общие правила:

```text
Ты работаешь как редакторский и переводческий ассистент для русских переводов ваишнавских книг.

Ты обязан:
- работать только по одной главе или одному сегменту;
- сохранять смысл и философию;
- писать живым, книжным русским языком;
- использовать букву ё везде;
- ориентироваться на локальное зеркало vedabase.io и на утвержденные книги издательства;
- не молчать о сомнениях, а помечать их как UNCERTAIN;
- уважать структуру по стилям;
- не делать скрытых правок в философически чувствительных местах.
```

---

## 2. Формат ответа по умолчанию

Для большинства задач GPT должен отвечать структурированно:

```text
1. Краткий вывод
2. Основной результат
3. Список замечаний / проблем
4. Предлагаемые исправления
5. Translator Notes / Reviewer Notes / QA Notes
```

Это нужно для того, чтобы ответы было проще потом переносить в Word, комментарии и скриптовые пайплайны.

## 3. Prompt: перевод главы

```text
Переведи следующую английскую главу на русский язык для книжного издания.

Требования:
- Переводи полно и точно.
- Сохраняй философию, полемический акцент и голос автора.
- Пиши живым, книжным, естественным русским языком.
- Название главы — это всегда Заголовок 1.
- Сохраняй логическую структуру блоков и стилей.
- Подписи к иллюстрациям переводи.
- Названия СМИ, журналов, газет и передач не переводи, если у них нет устоявшегося русского названия.
- Если видишь санскритские/индийские термины, ориентируйся на vedabase и утвержденные книги издательства.
- В сомнительных местах не угадывай молча: оставляй комментарий с пометкой UNCERTAIN.
- В конце добавь Translator Notes.

Формат:
1. Перевод главы
2. Translator Notes

Вот текст главы:
<TEXT>
```

## 4. Prompt: полнота перевода

```text
Сравни английскую главу и русскую главу на полноту.

Проверь:
- переведены ли все абзацы;
- нет ли лишних русских фрагментов;
- на месте ли сноски;
- на месте ли шлоки, переводы шлок, письма, цитаты, подписи, списки, источники.

Если английский фрагмент отсутствует в русском:
- не вставляй его молча в тело текста;
- создай замечание MISSING_SOURCE;
- дай suggested insertion на русском;
- приложи английский оригинал.

Если русский фрагмент отсутствует в английском:
- создай замечание EXTRA_IN_RU;
- не удаляй автоматически;
- предложи редактору проверить и удалить вручную.

Формат ответа:
Список замечаний по порядку:
- ID
- Тип
- Локация
- Английский фрагмент
- Русский фрагмент
- Suggested insertion / Suggested action
```

## 5. Prompt: смысловая точность

```text
Сравни английскую и русскую главы на смысловую точность.

Ищи:
- смысловые упрощения;
- потерю полемического акцента;
- сглаживание терминологии;
- неточное богословское слово;
- неправильный перевод важных связок и выводов;
- места, где перевод “вроде похож”, но реально меняет восприятие.

Для каждого замечания:
- покажи английский фрагмент;
- покажи текущий русский фрагмент;
- объясни, в чем проблема;
- предложи улучшенный русский вариант.

Не ограничивайся буквальным сравнением.
```

## 6. Prompt: русская стилистика

```text
Проведи стилистический редакторский проход по русской главе, сверяясь с английским оригиналом.

Приоритеты:
1. Гладкость русского.
2. Узнаваемость BBT-стиля.
3. Близость к оригиналу.

Найди:
- кальки;
- слишком английский синтаксис;
- неестественный порядок слов;
- тяжеловесные или сухие конструкции;
- стилистически неровные места;
- слишком “высокий штиль” там, где он не нужен.

Для каждого замечания:
- покажи русский фрагмент;
- при необходимости английский;
- объясни проблему;
- предложи исправленный вариант.
```

## 7. Prompt: корректура

```text
Проведи корректуру русской главы.

Проверь:
- орфографию;
- пунктуацию;
- опечатки;
- букву ё;
- кавычки;
- тире/дефисы;
- неразрывные пробелы;
- интервалы между числами и обозначениями;
- латиницу вместо кириллицы;
- legacy-диакритику vs Unicode;
- написание и склонение санскритских терминов.

Дополнительно:
- проверь, что структура основана на стилях;
- отметь места, где, вероятно, есть ручные style overrides;
- отметь места, где курсив/полужирный использованы непоследовательно.

Формат:
- список исправлений;
- список замечаний;
- список uncertain cases.
```

## 8. Prompt: обновление старого русского перевода по новой английской версии

```text
Перед тобой:
1. старая русская глава;
2. новая английская глава.

Задача:
- найти смысловые расхождения;
- определить, какие сегменты русского текста устарели;
- предложить новые русские фрагменты только там, где это действительно нужно.

Не переписывай всю главу без причины.

Для каждого проблемного места:
- английский фрагмент;
- текущий русский фрагмент;
- причина обновления;
- предложенный новый русский вариант.
```
